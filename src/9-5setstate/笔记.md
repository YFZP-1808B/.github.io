# setState
1.setState不会立刻改变React的组件中state的值
2.setState通过触发一次组件的更新来引发重绘
3.多次setState函数调用产生的效果会合并
# setState:异步
事件处理函数内部的setState是异步的
这种机制可以在大型应用中得到很好的性能提升
# 配置引用文件路径
  在项目一级目录里创建 jsconfig文件  写入:
  {
    "compilerOptions":{
        "baseUrl":"src"
    },
    "include": ["src"]
}
# 使用的时候可以绝对路径 src下的路径都可以用绝对路径
